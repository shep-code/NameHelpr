# Setup

## Prerequisites
1. Node.js v22+ (nodejs.org)
2. Claude Code: `npm install -g @anthropic-ai/claude-code`
3. GSD: `npx -y get-shit-done-cc@latest --global`

## Install
```bash
npm install
```

## Run
```bash
npm run dev
```

## Resume GSD work
cd into project, run `claude`, then `/gsd:progress`
